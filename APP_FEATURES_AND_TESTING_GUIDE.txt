================================================================================
                           1. APP OVERVIEW
================================================================================

DOLLARSMILEY is a marketplace app that connects:
- CUSTOMERS: People who need services
- PROVIDERS: People who offer services
- HYBRID: Users who both offer and need services

Key Capabilities:
- Browse and book services
- Create and manage listings
- Accept payments via Stripe
- Social community features
- Real-time messaging
- Advanced verification system
- AI-powered recommendations

================================================================================
                    2. USER ROLES & ACCOUNT TYPES
================================================================================

THREE ACCOUNT TYPES:
--------------------

1. CUSTOMER
   - Can browse services
   - Can book providers
   - Can make payments
   - Can leave reviews
   - CANNOT create listings
   - CANNOT receive payments

2. PROVIDER
   - Can create listings
   - Can accept bookings
   - Can receive payments
   - Must be verified
   - CANNOT book others (in provider mode)

3. HYBRID (Full Access)
   - All Customer features
   - All Provider features
   - Can switch contexts
   - Must be verified for provider features
   - Most flexible account type

ADMIN ROLE:
-----------
- Platform administrators
- Access to admin dashboard
- Can moderate content
- Can manage users
- Can view analytics
