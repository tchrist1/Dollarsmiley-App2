FILE UPLOAD ERROR - FIXED
=============================

ISSUE: "Cannot read property 'Base64' of undefined"
LOCATIONS: Edit Profile, Contact Provider Messages, All File Uploads

ROOT CAUSE:
-----------
Incompatible base64 conversion using 'base-64' package in React Native

SOLUTION:
---------
Replaced 'base-64' with 'base64-arraybuffer' in /lib/file-upload-utils.ts

CHANGED FUNCTION:
-----------------
fileUriToByteArray() - now uses base64-arraybuffer.decode()
